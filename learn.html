About Me


~ Manju Shettar ~

With a strong passion for Programming

Hey everyone! With this website, I aim to introduce Java concepts to people who have never coded before. Coding is a useful skill that everyone should attempt at least once in their life. By mastering Java, one can create: video games, highly-functional algorithms, input/output programs, etc. Java is a high-level, fast programming language that is used all over the world. If you have any questions, feel free to contact me.

~How to use this Course~

This course is fairly straightforward. If you are just beginning programming in Java or need to touch up on your skills, you should start at the bottom (Hello World) and move upwards. As the course progresses, keep in mind that skills from previous lessons may carry over. It's important that you practice to gain knowledge, not to finish the course. Make sure to be adamant. Some of these problems are tough and don't resort to online sources (i.e. StackOverflow) for help until you're absolutely on your last straw. Good luck!
