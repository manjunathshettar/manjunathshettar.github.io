While Loops

Let's say you want to decide what to do based on a condition. If the sun is out, we'll go outside for a walk. If not, we'll just stay inside and watch Netflix. We can use if-else blocks in Java to tackle this.

Create an integer, int value = 14;. Now type in: if(value < 15){. The if keyword denotes a if block. Now, after the open bracket, print out the value. Remember to close the block! Run the code and you should see that the value of value was printed to the terminal. This is because value is less than 15. If we change value to be 20 and run the code again, you'll see that value doesn't actually print. This is because it is now greater than 15.

Now type in this: else{ System.out.println("Your value is greater than or equal to 15!"); }. Run it again with value being 20. Notice what happens? If the conditionals doesn't hold true for the if block, the Thread will skip to the else block and run whatever happens there. You can have an if block without an else block, but you cannot have an else block without an if block.

Another part of the if-else concept is the else if. Let's keep the value of value at 20. Below the if block, but above the else block write else if(value > 15){System.out.println("Challenge completed!"); }. Since the value of value is 20, and thus greater than 15, we'll see that the code in our else if block is printed. The else if block will be printed if the code in the if block fails and the conditional in the else if block is true. You can have multiple else if blocks to test for all conditionals, but you can only have one if and else block.

Challenge Problem: Create a boolean and give it a value of false. Now, create an if-else structure that prints "Hello" if the boolean is true and "GoodBye" if the value of the boolean is false. Can you do this with only one if block and no else/else if's?
