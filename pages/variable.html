Variables and Declarations

Variables

Variables are nifty and fun. But what exactly is a variable? Well, in math, a variable is usually a placeholder for a number. In programming, variables can be used for numbers, words, letters, and conditionals! A variable is something programmers use to save time. Let's say I want to use the number 3.14159265359 in my program 20 times. I'm not going to type that number out 20 times! Instead, I'll store that number into a variable and type that variable name 20 times, which is bound to save me time.

So know that you know what we use variables for, let's get into it. Let's start of by declaring a number. Let's use the Integer class, a class that can only hold whole numbers. In your main method, type int variable;. We've done it! We've declared a variable. Print the variable out using this statement; System.out.println(variable);. What did you get? If you got zero, you're correct. Since we never specified the value for our variable, Java defaults our variable, and all Integers, to the value of zero. Let's change that.

Underneath the print statement, type variable = 5;. Print the value of variable one more time, and you should see that the new value of variable is 5. Amazing! This second step, where we actually assign a value is called initilization and the first step is called declaration. We can do this nifty trick for all future data types we learn!

Can we make this more efficient, though? The answer is yes! We can combine the declaration and initilziation into one step like this; int variable = 5;. Amazing right? Go ahead and create five more integer variables and print them to the terminal. Remember that you must have the "int" before the variable name to declare an Integer variable. Also note that the minimum and maximum values an Integer variable can handle is -2^31 and 2^31-1 respectively.
