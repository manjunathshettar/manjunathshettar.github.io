For Loops

Can you think of the easiest way to print out all of the numbers in range 1-40 to the terminal? Sure, you could use 40 different print statements and copy-paste the same System.out.println(num); 40 times. There's a simpler way, however, and it comes in the form of for loops.

Before explaining what a for loop actually does, let's look at the syntax of the strucutre. for(int i = 0; i < max-1; i++){ //do something }. Notice the lack of a semicolon after the header of the for loop. If you do add a semi-colon, you will experience many errors! Let's start with the first word; 'for'. This lets Java know that you are creating a for loop. The next part declares and initializes an integer variable with the value of 0. Let's think about this; we know that Java counts from 0 instead of 1, so it makes sense that our initial value would be 0.

The next part is the conditional part, or the boolean-part. The loop will continue to run as long as the second-part is true. The code reads; i < max-1 ;. This means that the code will keep running as long as i is less than one less than the maximum value. The conditional is read before the Thread actually goes into the loop, so make sure your coniditionals are accurate.

The last part is the incrementation part. We'll get into incrementation later, but for now, just know that Java is incrementing i by one everytime the loop completes. If we tie it all togther, we realize that i will eventually reach our max value, since we are incrementing i.

So, let's get printing! Let's print out all of the numbers in the 3 - 9 range. First; the header of the loop: for(int i = 3; i < 10; i++){. Did you get that? Since we are starting at 3, we have to initialize i to start at 3. Also, since we want to end at 9, we have to make sure that i is always less than 10. An alternate header would be; for(int i = 3; i<=9; i++){, the <= refers to less than or equal to.

Now, print i to the terminal by using .println();. Sit back and watch as your program spits out the numbers! You just created your first for loop and avoided writing over 9 lines of code!
